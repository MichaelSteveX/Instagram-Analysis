# Instagram Influencer Analysis (Python + Power BI Dashboard)

## Project Overview

This project focuses on analyzing Instagram influencer data to extract meaningful insights using Python for data analysis and Power BI for interactive dashboard visualization.

The goal of this project is to understand:

* Influencer engagement patterns
* Relationship between followers, likes, and comments
* Performance of different influencer categories
* Trends in audience interaction and growth
* Key factors affecting influencer success

This project demonstrates data analysis, visualization, and predictive modeling skills.

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Power BI

* Interactive Dashboards
* Data Modeling
* DAX
* Charts and Visualizations
* Filters and Slicers

## Dashboard Features

* Engagement Rate Analysis
* Followers vs Likes Comparison
* Top Performing Influencers
* Category-wise Analysis
* Trends in Likes and Comments

## Filters & Interactivity

* Influencer Category Filter
* Followers Range Filter
* Engagement Rate Filter
* Dynamic Slicers for exploration
